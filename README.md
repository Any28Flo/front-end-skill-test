

<h2 align="center">Desarollo de aplicación para Grut Studio</h2>


## Pre-requisitos


Tener instalado > NodeJS 8.x y NPM.

## Scripts disponibles

- Configurar variables .env 
- `npm install`: Instalar las dependencias

- `npm start` : Lanzar aplicación

## Tecnologías ocupadas 

- `axios, reactjs, hooks`
